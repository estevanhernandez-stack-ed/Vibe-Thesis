# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful. Disagreements are normal; personal attacks are not.
- Assume good faith. Most miscommunications aren't malicious.
- Give clear, actionable feedback. "This is wrong" isn't useful; "this fails because X — try Y" is.
- Make space for new contributors. Templates exist to be forked and customized; assume the asker is doing exactly that.
- Don't gatekeep. Academic / publishing background isn't a prerequisite to use or improve this template.

## Reporting

If you experience or witness behavior that violates this code, contact **conduct@626labs.dev**. Reports are confidential. Maintainers will respond within 72 hours.

## Enforcement

Maintainers may warn, temporarily ban, or permanently ban contributors who violate this code. Severity matches the violation.

For the full Contributor Covenant text, see <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
