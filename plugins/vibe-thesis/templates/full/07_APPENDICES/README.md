# 07_APPENDICES

Supplementary materials that support but don't fit the main body — extended derivations, full datasets, supplementary figures, methodological details that would interrupt the argument flow.

## Render behavior

Files in this directory are appended to the rendered output AFTER the main body and bibliography, in numeric filename order. Each appendix gets its own labeled section (`Appendix A`, `Appendix B`, ...).

## Mode notes

| Mode | Typical use |
| --- | --- |
| `dissertation` | Extensive — full data tables, additional analyses, raw source excerpts |
| `article` | Brief — supplementary figures, extended methods (also called "Supplementary Materials" in journal context) |
| `masters` | Moderate — full data, methodological details |

## File naming

Use `A-<descriptor>.md`, `B-<descriptor>.md`, etc. Renders pick up the letter prefix as the appendix label.
