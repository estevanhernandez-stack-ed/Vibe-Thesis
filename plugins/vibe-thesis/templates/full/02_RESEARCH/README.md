# 02_RESEARCH

Raw research notes, organized into axis-specific subdirectories so research-phase agent swarms can ingest in parallel.

## Default axes

| Subdirectory | Purpose |
| --- | --- |
| `prior-art/` | What's been done in your area; gap analysis. |
| `methodology-survey/` | How others have approached the question; method comparison. |
| `opposing-positions/` | The strongest case AGAINST your thesis. (Address these in-text per pillar 3.) |
| `key-authors/` | Author-centric notes on field leaders. |
| `primary-sources/` | Direct engagement with primary materials. |

These axes match the default `lit-review-swarm` playbook in `04_AGENT_SWARMS/`. Add or rename axes to fit your topic — the swarm playbook should be updated to match.

## Structure within each axis

Free-form. Markdown notes, scanned PDFs (in `_pdfs/` subdirs), excerpt files. The swarm convergence step synthesizes these into a lit-review skeleton you then write up in `03_BODY/`.

## Hand-research vs. swarm

For narrow, deep dives (re-reading a single canonical text), hand-research. For broad coverage across multiple axes (initial lit review, finding opposing positions), dispatch a swarm. See `04_AGENT_SWARMS/README.md` for the decision rule.
